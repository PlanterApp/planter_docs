Planter now supports intercropping different plants in the same garden square!

Note: intercropping is only available in the new Gardens 2.0 garden format. Convert your existing garden plans to be able to use intercropping.

{{< video mp4-src="recording.mp4" attributes="controls muted playsinline preload=metadata" >}}

1. ## **Select a Plant to Intercrop With**    **{{< screenshot 01-select-garden-section.webp "Select Garden Section" >}}**    Tap on a plant in the garden to bring up the action menu.

2. ## **Choose Intercrop Option**    **{{< screenshot 02-choose-intercrop-option.webp "Choose Intercrop Option" >}}**    From the circular action menu, tap the "Intercrop" option.

3. ## **Select a Plant to Intercrop**    **{{< screenshot 03-select-onions-plant.webp "Select Onions Plant" >}}**    In the Select plant screen, locate and select the plant you wish to add as an intercrop plant.

4. ## **Intercropping With Larger Multi-Square Plants**    **{{< screenshot 04-place-basil.webp "Place Basil" >}}**    To intercrop plants around a larger, multi-square plant (e.g., tomatoes), repeat the steps above then select where the intercrop plant should be added.
