Planter now lets you plan garden beds individually then combine them together in one combined view!

Note: The ability to create multiple garden beds is a Planter Premium feature.

1. **Create a New Garden**

   {{< screenshot 01-initiate-new-garden-creation.webp "Initiate New Garden Creation" >}}

   Tap the 'NEW GARDEN' button located at the bottom right of the Gardens screen. This new garden will be the combined view garden.
2. **Select New Garden Planner**

   {{< screenshot 02-select-legacy-garden-planner.webp "Select Legacy Garden Planner" >}}

   From the 'Create new garden' dialog, select 'Try new garden planner'.
3. **Enter Garden Name**

   {{< screenshot 03-enter-garden-name.webp "Enter Garden Name" >}}

   Type a name for the combined view garden into the 'Garden name' text field at the top of the screen.
4. **Adjust Garden Width and Height**

   {{< screenshot 04-adjust-garden-width.webp "Adjust Garden Width" >}}

   Leave the garden shape as a standard rectangular shape and drag the 'Width' and ‘Height’ sliders to set the garden width and height (consider the widths and heights of the gardens you wish to combine).

5. **Create Garden**

   {{< screenshot 05-create-garden.webp "Create Garden" >}}

   Tap the 'CREATE' button at the bottom right of the screen to finalize the new garden creation.
6. **Open the “Three Dot” Options Menu**

   {{< screenshot 06-open-options-menu.webp "Open Options Menu" >}}

   Tap the three vertical dots icon in the top right corner of the screen to open the garden options menu.
7. **Add Garden Bed**

   {{< screenshot 07-add-garden-bed.webp "Add Garden Bed" >}}

   From the dropdown menu, select 'Add garden bed' to add a garden bed to the current view.
8. **Select a Garden Bed to Add**

   {{< screenshot 08-select-rounded-garden-bed.webp "Select Rounded Garden Bed" >}}

   In the 'Choose a garden bed' dialog, select a garden bed to add from the list of available garden beds (or, choose “Create new garden” to create a new bed to add). **Note:** only garden beds in the new Gardens 2.0 format can be combined in one view.
9. **Set Row and Column Position**

   {{< screenshot 09-set-row-position-for-garden-bed.webp "Set Row Position for Garden Bed" >}}

   Drag the 'Row' and ‘Col’ sliders to set the desired position for the new garden bed in the combined garden view.

10. **Place Garden Bed**

    {{< screenshot 10-place-rounded-garden-bed.webp "Place Rounded Garden Bed" >}}

    Tap the 'Place garden bed' button to add the garden bed to the combined garden view.

11. **Repeat for Additional Beds**

    {{< screenshot 11-repeat-for-additional-beds.webp "Repeat for Additional Beds" >}}

    Repeat the steps above to add additional beds to the garden view. A warning will appear if the beds are overlapping.

12. **Move a Bed**

    {{< screenshot 12-move-a-bed.webp "Move a Bed" >}}

    To move a bed in the combined garden view, long-press on the bed then select ‘Move’. Drag the sliders to the new row and column position then tap ‘Move garden bed’.

**13. Remove a Bed**

{{< screenshot 12-move-a-bed.webp "Remove a Bed" >}}

To remove a bed from the combined garden view, long-press on the bed then select “Remove from garden”. The garden bed will still be visible in the Gardens list as an individual bed.
