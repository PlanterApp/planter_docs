Planter now supports row spacing and “triangular” spacing, in addition to per-square-foot spacing.

{{< video mp4-src="recording.mp4" attributes="controls muted playsinline preload=metadata" >}}

1. **Select Planting Area**

   {{< screenshot 01-select-planting-area.webp "Select Planting Area" >}}

   Tap the top-left corner of the desired planting area, then tap the bottom-right corner to define a rectangular space to plant.
2. **Open Add Plants Menu**

   {{< screenshot 02-open-add-plants-menu.webp "Open Add Plants Menu" >}}

   Tap the "Add Plants" button located at the bottom of the screen.
3. **Choose Row Planting**

   {{< screenshot 03-choose-row-planting.webp "Choose Row Planting" >}}

   From the "Add Plants" dialog, select "Row Planting".
4. **Select Plant Type**

   {{< screenshot 04-select-plant-type.webp "Select Plant Type" >}}

   On the "Add Row Planting" screen, tap the pencil icon next to "Select plant" to choose a plant.
5. **Choose Onions**

   {{< screenshot 05-choose-onions.webp "Choose Onions" >}}

   Scroll through the plant list to select a plant for row planting.
6. **Set Between-Row Spacing**

   {{< screenshot 06-set-row-spacing.webp "Set Row Spacing" >}}

   Tap a spacing option under "Row Spacing" to set the distance between rows (or select ‘Custom’ to define a custom between-row spacing).
7. **Set In-Row Plant Spacing**

   {{< screenshot 07-set-row-plant-spacing.webp "Set Row Plant Spacing" >}}

   Tap a spacing option under "Row Plant Spacing" to set the distance between plants in the row (or select ‘Custom’ to define a custom in-row spacing).
8. **Create Row Planting**

   {{< screenshot 08-create-row-planting.webp "Create Row Planting" >}}

   Tap the "Create" button to finalize and add the row planting to your garden layout.
